## Installation

### Composer package
Use composer to install this package.

```
{
    "require": {
        "pio-przybysz-api": "*"
    }
}
```


## Basic usage 

```
 $api = new PioApi('<login>', '<password>');
```

## Tests

To run tests
```
composer test
```
